Генератор штрихкодов

Можно ввести 12 цифр для EAN-13 (или 7 для EAN-8) — контрольная цифра будет добавлена автоматически; либо полный код с уже верной контрольной цифрой.

В международном формате, со знаком + — так надёжнее для сканеров.

Результат

Изображение кода появится здесь после нажатия «Сгенерировать».
